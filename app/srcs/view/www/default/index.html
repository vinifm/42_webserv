HTTP STATUS RESPONSE

7

404 Not Found

This page is presented in browser when the URL you type don't correspond to any route, file or directory on server.




TESTAR
8

403 Forbidden

It occurs when try to access one URL and don't have permission. (For example, try to access directory in some route and don't have autoindex allowed for that directory.)


TESTAR
9

400 Bad Request

It occurs when you send a request that is incomplete or has invalid parameters.


TESTAR
7

500 Internal Server

The server encountered an unexpected error while processing the request.




TESTAR
8

503 Service Unavailable

The server is temporarily unable to handle the request, typically due to overload or maintenance.
